Before setting up the SQL Data Warehouse project, ensure you have the following prerequisites in place.

## Docker Environment

<Card title="Docker & Docker Compose" icon="docker" iconType="duotone">
  This project runs entirely in Docker containers, making it easy to set up and tear down without affecting your local system.
</Card>

<AccordionGroup>
  <Accordion title="Docker Desktop" icon="desktop">
    Install Docker Desktop for your operating system:
    - **Windows**: [Docker Desktop for Windows](https://docs.docker.com/desktop/install/windows-install/)
    - **macOS**: [Docker Desktop for Mac](https://docs.docker.com/desktop/install/mac-install/)
    - **Linux**: [Docker Engine](https://docs.docker.com/engine/install/)
  </Accordion>

  <Accordion title="Docker Compose" icon="layer-group">
    Docker Compose is included with Docker Desktop. For Linux users, you may need to install it separately:
    ```bash
    sudo apt-get install docker-compose-plugin
    ```
  </Accordion>
</AccordionGroup>

## System Requirements

<CardGroup cols={2}>
  <Card title="Memory" icon="memory">
    Minimum 4GB RAM available for Docker
  </Card>
  <Card title="Storage" icon="hard-drive">
    At least 2GB free disk space
  </Card>
  <Card title="CPU" icon="microchip">
    2+ CPU cores recommended
  </Card>
  <Card title="OS" icon="computer">
    Windows 10+, macOS 10.15+, or Linux
  </Card>
</CardGroup>

## Required Knowledge

<Steps>
  <Step title="PostgreSQL Basics">
    Understanding of PostgreSQL databases, SQL queries, and basic database administration.
    
    <Tip>Familiarity with concepts like schemas, tables, and views will be helpful.</Tip>
  </Step>

  <Step title="Docker Fundamentals">
    Basic knowledge of Docker containers and Docker Compose for managing multi-container applications.
  </Step>

  <Step title="Data Warehousing Concepts">
    Understanding of ETL processes, data modeling, and the Medallion Architecture (Bronze, Silver, Gold layers).
  </Step>
</Steps>

## Dataset Requirements

<Info>
  This project requires CSV datasets from two source systems: **ERP** and **CRM**.
</Info>

The datasets should include:

<CardGroup cols={2}>
  <Card title="ERP Data" icon="database">
    Enterprise Resource Planning system data containing:
    - Sales transactions
    - Product information
    - Operational data
  </Card>
  
  <Card title="CRM Data" icon="users">
    Customer Relationship Management data containing:
    - Customer information
    - Interaction history
    - Customer demographics
  </Card>
</CardGroup>

<Warning>
  Ensure your CSV files are properly formatted and placed in the `datasets/` directory before starting the setup process.
</Warning>

## Verification Checklist

Before proceeding to installation, verify that you have:

<Check>Docker Desktop installed and running</Check>
<Check>Docker Compose available (check with `docker compose version`)</Check>
<Check>Sufficient system resources allocated to Docker</Check>
<Check>CSV datasets ready in the datasets directory</Check>
<Check>Basic understanding of PostgreSQL and data warehousing</Check>

<Note>
  Once all prerequisites are met, you can proceed to the [Installation](/setup/installation) guide.
</Note>
